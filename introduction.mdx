## What is Cymbal Shops?

Cymbal Shops (also known as Online Boutique) is a cloud-first microservices demo application built by Google. It demonstrates how to design, build, and operate a production-grade e-commerce platform on Google Cloud using modern cloud-native technologies.

The application is a fully functional web-based shop where users can browse products, add items to a cart, and complete purchases — all backed by 11 independent microservices communicating over gRPC.

## Microservices overview

Cymbal Shops is composed of 11 microservices written in different languages:

| Service | Language | Role |
|---|---|---|
| frontend | Go | Serves the web UI and HTTP API |
| cartservice | C# | Manages shopping cart state in Redis |
| productcatalogservice | Go | Product listings and search |
| currencyservice | Node.js | Currency conversion |
| paymentservice | Node.js | Payment processing (mock) |
| shippingservice | Go | Shipping quote and order fulfillment (mock) |
| emailservice | Python | Order confirmation emails (mock) |
| checkoutservice | Go | Orchestrates the full checkout flow |
| recommendationservice | Python | Product recommendations |
| adservice | Java | Contextual ads |
| loadgenerator | Python/Locust | Simulates realistic user traffic |

<Note>
  This is a demonstration application. Payment, shipping, and email services are mocked and do not process real transactions.
</Note>
