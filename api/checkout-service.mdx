## Overview

`CheckoutService` is the central coordinator for purchases. When a user places an order, the checkout service retrieves their cart, converts prices to the user's preferred currency, charges the credit card via `PaymentService`, arranges shipment via `ShippingService`, and sends a confirmation email via `EmailService`.

```protobuf
service CheckoutService {
    rpc PlaceOrder(PlaceOrderRequest) returns (PlaceOrderResponse) {}
}
```

---

## PlaceOrder

Executes a complete purchase for the user. This is an orchestrating RPC that calls multiple downstream services internally.

### Request — `PlaceOrderRequest`

<ParamField body="user_id" type="string" required>
  The unique identifier of the user placing the order. Used to retrieve the cart.
</ParamField>

<ParamField body="user_currency" type="string" required>
  The 3-letter ISO 4217 currency code for displaying prices (e.g., `"USD"`, `"EUR"`).
</ParamField>

<ParamField body="address" type="Address" required>
  The shipping address for the order.

  <Expandable title="Address properties">
    <ParamField body="street_address" type="string" required>
      Street number and name.
    </ParamField>
    <ParamField body="city" type="string" required>
      City name.
    </ParamField>
    <ParamField body="state" type="string" required>
      State or province.
    </ParamField>
    <ParamField body="country" type="string" required>
      Country name.
    </ParamField>
    <ParamField body="zip_code" type="int32" required>
      Postal or ZIP code.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="email" type="string" required>
  The email address to send the order confirmation to.
</ParamField>

<ParamField body="credit_card" type="CreditCardInfo" required>
  The payment card details.

  <Expandable title="CreditCardInfo properties">
    <ParamField body="credit_card_number" type="string" required>
      The credit card number as a string (e.g., `"4432-8015-6152-0454"`).
    </ParamField>
    <ParamField body="credit_card_cvv" type="int32" required>
      The 3- or 4-digit card verification value.
    </ParamField>
    <ParamField body="credit_card_expiration_year" type="int32" required>
      The 4-digit expiration year (e.g., `2030`).
    </ParamField>
    <ParamField body="credit_card_expiration_month" type="int32" required>
      The expiration month as an integer from `1` to `12`.
    </ParamField>
  </Expandable>
</ParamField>

<ResponseField name="order" type="OrderResult">

    </ResponseField>
    <ResponseField name="shipping_address" type="Address">
      The address the order was shipped to.

      <Expandable title="Address properties">
        <ResponseField name="street_address" type="string">

        </ResponseField>

        <ResponseField name="state" type="string">

          Country name.
